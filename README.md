# Jacob Vierneza Golf Recruiting Website

## GitHub Pages Setup

1. Create a new GitHub repository, for example: `jacob-vierneza-golf`
2. Upload these files:
   - `index.html`
   - `styles.css`
3. In GitHub, go to **Settings > Pages**
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. Your site will publish at:
   `https://YOUR-GITHUB-USERNAME.github.io/jacob-vierneza-golf/`

## Before Sharing With Coaches

Update these placeholders:
- Player email
- Player phone
- Parent phone
- Swing video links
- Resume PDF link
- Upcoming tournament schedule
- Coach/reference contact
